ZepraBrowser Development TODO

⚠ BETA v0.1.0 BLOCKER — Updated 2026-06-12
The single largest gap preventing modern site compatibility is the header-only JSDOMBridge (source/webCore/scripting/js_dom_bridge.hpp). There is no .cpp implementation and no concrete DOM classes (DOMElement, DOMDocument, etc.).

Result: Static HTML/CSS pages render correctly. Any page relying on JavaScript (React, Next.js, Vue, Angular, SPAs) renders as a blank or broken shell because the DOM never materializes.

See the new detailed reports: beta-css-status.html — CSS is production-grade but blocked by missing DOM.
js-dom-bridge-gap.html — Root cause analysis and recommended path forward.
