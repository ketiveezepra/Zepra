🌐 Zepra Browser

v0.1.0-beta
The first public beta of an independent web browser built entirely from scratch by KetiveeAI.
⚠ Beta Software Notice

This is a pre-release version intended for testing and community feedback. It may contain bugs, incomplete features, and performance issues. Do not use this as your primary browser for sensitive activities (banking, medical, legal).

What's Inside

⚡

ZepraScript Engine

Custom JavaScript engine with multi-tier JIT, ES2023+ support, and WebAssembly interpreter.

🎨

NXRender

Hardware-accelerated OpenGL rendering with HTML5/CSS3 layout and NXSVG vector graphics.

🌐

NXNet

Custom HTTP client, TLS/SSL, DNS resolver, cookie manager, and response cache.

🔒

Security First

File quarantine, sandbox flags, executable detection, and URL scheme classification.

🗂️

Tab Management

Tabbed browsing with tab suspension for memory management and per-tab history.

🤖

AI Sidebar

Page summarization, Q&A, and entity extraction powered by ZepraSearch ML services.

Installation

🪟 Windows
🐧 Linux
🍎 macOS

Option A: GUI Installer (Recommended)

Download the installer package and run:

# Right-click and select "Run with PowerShell"
Install-ZepraBrowser.ps1

# Or from terminal (as Administrator):
powershell -ExecutionPolicy Bypass -File Install-ZepraBrowser.ps1

Option B: Silent Install

powershell -ExecutionPolicy Bypass -File Install-ZepraBrowser.ps1 -Silent

System Requirements: Windows 10 or later, x86_64, 2 GB RAM, OpenGL 2.0

Option A: .deb Package (Debian/Ubuntu)

# Install the .deb package
sudo apt install ./zepra-browser_0.1.0-beta_amd64.deb

# Or with dpkg:
sudo dpkg -i zepra-browser_0.1.0-beta_amd64.deb
sudo apt-get install -f  # fix dependencies

Option B: Interactive Installer

chmod +x install-zepra.sh
sudo ./install-zepra.sh

The installer auto-detects GUI availability and uses zenity for graphical dialogs when available.

Dependencies: libfreetype6, libx11-6, libgl1, zlib1g, libssl3

DMG Installation

1. Download ZepraBrowser-0.1.0-beta.dmg

2. Double-click to mount the disk image

3. Drag Zepra Browser to the Applications folder

4. Right-click and select "Open" on first launch (unsigned beta)

Requirements: macOS 11.0 (Big Sur) or later
